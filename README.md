## Structure

- `index.html` — page personnelle (ex 2.1)
- `css/` — feuilles de style (ex 3.1)
- `js/` — scripts JavaScript, cartes interactives + chatbot (ex 6.1 et 7.2)
- `pages/` — pages secondaires (cartes, chatbot, formulaire, sources)
- `php/` — traitement serveur du formulaire (ex 5.1)
- `sql/` — schéma de la base MySQL (ex 5.1)
- `images/` — visuels utilisés dans le site
