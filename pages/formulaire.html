Exercice 5.1, PHP et MySQL

Rejoindre la newsletter

Le formulaire ci-dessous est traité par un script PHP qui enregistre les inscriptions dans une base MySQL hébergée chez AlwaysData. Cinq champs, validation côté serveur, requêtes préparées PDO, mot de passe stocké hors du dossier www.

01 Inscription

Entre 3 et 40 caractères, c'est le nom qu'on verra sur le leaderboard.
Pour recevoir les annonces de tournois et les sorties de nouveaux sets. Pas de revente, jamais.
On t'envoie les nouveautés en priorité sur le set que tu préfères.
Optionnel, 600 caractères maximum.

02 Comment ça fonctionne côté serveur

Quand tu valides le formulaire, le navigateur envoie une requête HTTP POST au fichier php/inscrire.php. Le serveur Apache d'AlwaysData reçoit la requête et la délègue au moteur PHP 8. Le script PHP fait alors trois choses :

  1. Il vérifie que tous les champs sont présents et que l'email a une forme valide via la fonction filter_var.
  2. Il insère la nouvelle inscription dans la table inscriptions de la base MySQL matteocb-uor_naruto, via une requête préparée PDO pour éviter toute injection SQL.
  3. Il renvoie une page de confirmation HTML avec un résumé de l'inscription, ou un message d'erreur si quelque chose n'a pas marché.

Le schéma de la table et le code PHP complet sont expliqués dans le PDF rendu pour le cours.
