Bibliographie

Sources utilisées

Liste complète des références utilisées pour la rédaction des pages HTML, du CSS, du JavaScript et du backend PHP/SQL de ce site, dans le cadre du cours UOR à l'IED Paris 8.

01 Documentation officielle

  • MDN Web Docs, Mozilla Developer Network, références HTML, CSS et JavaScript, developer.mozilla.org. Consulté entre mai et juin 2026 pour les sélecteurs CSS, les événements DOM et l'API Fetch.
  • Manuel PHP officiel, fonctions filter_var, htmlspecialchars, classe PDO, php.net.
  • MySQL 8.0 Reference Manual, syntaxe SQL et types ENUM, dev.mysql.com.
  • W3C Markup Validation Service, validateur HTML utilisé pour valider chaque page du site avant rendu, validator.w3.org.
  • Specification HTML Living Standard, WHATWG, pour la sémantique des balises nav, main, article, section, html.spec.whatwg.org.

02 Ressources d'apprentissage

  • W3Schools, tutoriels HTML, CSS, JavaScript et PHP, w3schools.com. Utilisé en complément de MDN pour quelques exemples concrets.
  • OpenClassrooms, support de la formation Développeur intégrateur web suivie en 2023, openclassrooms.com.

03 Métadonnées et standards

  • Dublin Core Metadata Initiative, schéma Dublin Core Elements 1.1, dublincore.org.
  • Open Graph Protocol, Facebook, ogp.me.
  • Twitter Cards, documentation X, developer.x.com.

04 Source du projet personnel

  • Projet Naruto Mythos Game, code source du simulateur en ligne dont je suis l'auteur, hébergé sur narutomythosgame.com. Les images des cartes et le principe de l'effet holographique CSS sont repris de ce projet.
  • Cicaboom, éditeur italien officiel du jeu Naruto Mythos TCG, cicaboom.com. Source des informations sur les sets, les rarités et les règles du jeu.
  • Site officiel Naruto TCG Mythos, page produit en français, narutotcgmythos.com/fr. Source officielle pour les visuels marketing du jeu, les annonces de sorties et la liste des revendeurs.
  • TopDeck.gg, agrégateur de tournois TCG, topdeck.gg. Utilisé pour les références aux tournois physiques officiels.

05 Cours et documents pédagogiques

  • P. Kislin-Duval, Informatique Fondamentale, partie 2, Cours UOR, support PDF de Licence 1 Informatique, IED Université Paris 8, version 2025-2026.
  • IED Université Paris 8, site institutionnel, iedparis8.net.
